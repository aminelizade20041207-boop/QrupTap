# **App Name**: ClassTrack

## Core Features:

- Class Detail Entry: Allows users to input and edit class details including name, time, and location.
- Schedule Views: Provides daily and weekly schedule views for users to easily see their classes.
- Offline Notifications: Sends notifications 5 minutes before each class starts, even when offline. Notification includes class name and room number.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to promote focus and tranquility.
- Background color: Light gray (#F5F5F5) for a clean and unobtrusive background.
- Accent color: A slightly darker blue (#42A5F5) is used for interactive elements and important information, providing contrast against the light gray background.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a clean, modern, neutral look that enhances readability for schedules and class details.
- Simple, minimalist icons representing different class types or subjects for visual clarity.
- Clean, structured layout that is optimized for quick viewing of schedule information and ease of navigation.
- Subtle transitions and animations when switching between daily and weekly views.
